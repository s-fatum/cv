### 🚀 Профессиональный профиль

Инженер с опытом более 8 лет в разработке сложных систем. Специализируюсь на построении высоконагруженных бэкендов на PHP 8.3 и интерактивных фронтенд-решений. Мой основной фокус — **iGaming** и **Fintech**, где критически важна целостность данных, высокая производительность и сложная транзакционная логика.

- 🎨 **Frontend:** Vue 3, TypeScript, GSAP, PixiJS (GameDev).
- 🤖 **AI-Driven Dev:** Активно использую AI-агентов для валидации архитектуры, рефакторинга и ускорения поставки качественного кода.
